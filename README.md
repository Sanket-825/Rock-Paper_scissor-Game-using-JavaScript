# Rock-Paper_scissor-Game-using-JavaScript
# Rock-Paper-Scissors Game
# This repository contains a simple JavaScript implementation of the classic Rock-Paper-Scissors game.
# Features:
# 1.User vs. Computer: Play against the computer in a friendly game of Rock-Paper-Scissors.
# 2.Score Keeping: Track your wins and the computer's wins with a dynamic score display.
# 3.Result Feedback: Receive immediate feedback on each round, including the winner and the reason for the outcome.
# 4.Reset Button: Easily reset the game to start a new match with a clean score.
